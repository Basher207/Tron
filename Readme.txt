Due to depreciated library causing bugs on different systems, joystick controller is not implemented in this version, note that its possible to use a joystick to button implementation to make them work. 


Controls:

ESC - Game pause

Player 1:
Up - Up arrow
Down - Down arrow
Right - Right arrow
Left - Left arrow

Player 2:
Up - W
Down - S
Right - D
Left -A

Player 3:
Up - T
Down - G
Right - H
Left - F

Player 4:
Up - I
Down - K
Right - L
Left - J

Power Ups:

Speed Up - Speeds yourself up for 2 seconds
Tunnel - Allows you to go underground for 2 seconds
Blast - Creates a gap in the lines in front of you on pick up
